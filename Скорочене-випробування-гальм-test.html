Тести для самоконтролю
Тема: Скорочене випробування гальм (МВРС)

Інструкція: Виконайте три блоки тестових завдань для перевірки знань з теми скороченого випробування гальм на моторвагонному рухомому складі. Натисніть кнопку "Показати відповіді" після завершення.

Тест 1: Вибір правильної відповіді

1. У якому випадку проводиться скорочене випробування гальм замість повного?
2. Який мінімальний тиск у головних резервуарах необхідний для проведення скороченого випробування?
3. Яка мінімальна напруга в ланцюзі ЕПТ допускається у хвостовому вагоні при скороченій перевірці?
4. На скільки необхідно знизити тиск у ГМ при скороченому випробуванні для перевірки гальмування?
5. Яким приладом контролюють дію гальм хвостового вагона при скороченому випробуванні?

Тест 2: Заповнення пропусків

1. При скороченому випробуванні після зміни бригади відстой поїзда не повинен перевищувати годин.
2. Для перевірки цілісності гальмівної магістралі використовують крана.
3. На дизель-поїзді серії Д у робочій кабіні пакетний перемикач встановлюють у положення .
4. При скороченому випробуванні не перевіряється ГМ за часом, якщо не було розриву магістралі.
5. Результати скороченого випробування оформляють у журналі форми .

Тест 3: Правильно/Неправильно

1. Скорочене випробування проводиться при відстої поїзда понад 3 години.
2. При зміні кабіни керування виконується скорочене випробування гальм.
3. При скороченому випробуванні перевіряється щільність ГМ шляхом виміру падіння тиску за 1 хвилину.
4. Скорочене випробування можна проводити після причеплення до поїзда більше 50% вагонів.
5. При скороченому випробуванні перевіряється напруга в ланцюзі ЕПТ не менше 45 В у робочій кабіні.

Правильні відповіді:

Тест 1: Вибір правильної відповіді

1. В — При зміні кабіни керування (а також при зміні бригади при відстої до 3 год, причепленні/відчепленні вагонів)
2. В — 7.5-9.0 кгс/см² (0.75-0.90 МПа)
3. Б — 30 В (у робочій кабіні не менше 45 В)
4. Б — 0.5-0.6 кгс/см² (0.05-0.06 МПа)
5. Б — Манометром гальмівного циліндра або лампою ГХВ (гальмо хвостового вагона)

Тест 2: Заповнення пропусків

1. 3 (три)
2. перекриття (закриття)
3. "Голова поїзда"
4. щільність
5. ТУ-152

Тест 3: Правильно/Неправильно

1. Неправильно (при відстої понад 3 год проводиться повне випробування)
2. Правильно
3. Неправильно (при скороченому випробуванні щільність ГМ не перевіряється, якщо не було розриву)
4. Неправильно (при причепленні більше 50% вагонів проводиться повне випробування)
5. Правильно
