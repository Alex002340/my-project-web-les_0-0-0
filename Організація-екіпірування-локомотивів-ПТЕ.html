Що таке екіпірування?

Визначення та суть процесу

Визначення

Екіпірування — це комплекс операцій щодо постачання локомотивів необхідними матеріалами для забезпечення їх працездатності під час експлуатації.

Примітка: Екіпірування поєднується з технічним обслуговуванням ТО-2 і виконується на закритих позиціях депо.

Обладнання локомотивів

Вимоги ПТЕ до пристроїв безпеки та екіпірування

Пристрої автоматичної зупинки

Кожний пульт управління пасажирським локомотивом, локомотивом, що обслуговується одним машиністом, та моторвагонним поїздом мусить мати пристрій автоматичної зупинки на випадок раптової втрати машиністом здатності вести поїзд.

Примітка

Пристрій автоматичної зупинки — це система безпеки, що активується при втраті машиністом свідомості або здатності керувати поїздом.

Маневрові локомотиви

Відчеплення з кабіни

Маневрові локомотиви мають обладнуватися пристроями для відчеплення їх від вагонів з кабіни машиніста.

Другий пульт управління

Локомотиви, що обслуговуються одним машиністом, крім того, мають другий пульт управління сигнальними лампами з обох сторін на зовнішньому боці кабіни.

Дзеркала заднього виду

Обов'язкове обладнання локомотивів, що обслуговуються одним машиністом, для забезпечення огляду за ходом поїзда.

Додаткові пристрої безпеки

Поїзні локомотиви, які обслуговуються одним машиністом, повинні мати додаткові пристрої безпеки руху, визначені Державною адміністрацією залізничного транспорту України.

Важливо: Перелік додаткових пристроїв безпеки встановлюється централізовано для всіх залізниць України.

Види екіпірування

Технічні засоби та обладнання пунктів екіпірування

Стаціонарні пристрої

Мають постійне технічне оснащення, розташоване на території депо або в безпосередній близькості.

  • Екіпірувальні колії в депо
  • Закриті позиції ТО-2 з екіпіруванням
  • Пункти технічного обслуговування (ПТО)
  • Суміщені пристрої для поїзних і маневрових локомотивів

Екіпірувальні пристрої повинні забезпечувати екіпірування одночасно не менше двох локомотивів на суміжних коліях.

Пересувні пристрої

Автоекіпірувальники на базі вантажних автомобілів для екіпірування локомотивів на ділянкових станціях.

  • Заправка дизельним паливом
  • Подача піску
  • Мастильні та обтиральні матеріали
  • Скорочення непродуктивних пробігів

Особливо вигідні при значному віддаленні від стаціонарних пунктів або завантаженості станційних колій.

Піскопостачання локомотивів

Особливості організації подачі піску

Механізація подачі

Подача піску на локомотиви повинна бути механізована. Застосовуються піскороздавачі з бункерами для сухого піску.

Зимовий запас

Зберігання запасів сухого піску для роботи взимку необхідно передбачати в закритих складах місткістю, що дорівнює 6-місячній витраті піску локомотивами.

Піскосушарки

Потужність піскосушарок встановлюється з розрахунку споживання піску для поточної роботи та створення зимового запасу на період припинення роботи кар'єрів.

Особливості для електровозів

Для електровозів площадки для подачі піску необхідно розташовувати:

  • На рівні дахів локомотивів
  • Одночасно для огляду струмоприймачів
  • З пристроями для зняття/подачі напруги
  • З необхідною сигналізацією та блокуванням

Добір піску на коліях

Пристрої для добору піску можливо передбачати на приймально-відправних коліях для оперативного поповнення запасів.
