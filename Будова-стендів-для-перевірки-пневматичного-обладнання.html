Стенд для авторежимів

Стенд уніфікованої конструкції призначений для випробування авторежимів гальмування. Забезпечує створення заданих тисків та темпів їх зміни для перевірки характеристик пристрою.

20 л

Ключові елементи схеми:

  • • Привалковий фланець з притиском для кріплення авторежиму
  • • Дроселі: ДР1 (3 мм) для гальмування/відпустки, ДР2 (1 мм) для штучного витоку
  • • Задатчик тиску для створення 3,0±0,1 та 4,2±0,1 кгс/см²
  • • Механізм зміни положення упору (295±1 мм від осі отворів)

Контрольні запитання

1

Призначення стенду уніфікованої конструкції?

Для випробування авторежимів гальмування з контролем усіх режимів роботи та характеристик.

2

Які резервуари входять до складу стенду реле тиску?

Резервуари об'ємами 20 л (Р10-20) та 55 л (Р10-55).

3

Призначення дроселів ДР1 та ДР2 на стенді авторежимів?

ДР1 (3 мм) – для створення темпу гальмування та відпустки; ДР2 (1 мм) – для штучного витоку повітря.

4

Який мінімальний тиск потрібен для роботи стенду повітророзподільників?

Не нижче 0,6 МПа (6 кгс/см²) у напірній магістралі.
