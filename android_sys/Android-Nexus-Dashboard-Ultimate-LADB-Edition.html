ℹ️ LADB & ADB Shell
🔌 Что такое LADB?
LADB (Local ADB) позволяет выполнять ADB-команды прямо на устройстве без компьютера. Требует Android 10+ и Wireless Debugging.

Различия команд:
• LADB — команды shell, работают в терминале LADB
• CMD — системные утилиты Android (cmd package, cmd netpolicy)
• Settings — изменение базы данных настроек
⚠️ Безопасность: Некоторые команды (особенно с --force-stop или очисткой кэша) могут нарушить работу приложений. Применяйте с пониманием.
🔌 LADB: Подключение
  • adb pair IP:PORT CODE
    Android 11+ Первичное сопряжение Wireless Debugging
  • adb connect IP:PORT
    Подключение после сопряжения
  • adb shell sh /sdcard/Android/data/com.draco.ladb/files/press_ssh.sh
    Автоматизация подключения LADB (скрипт в папке приложения)
  • adb pair localhost:PORT CODE
    Split-screen Сопряжение в режиме разделённого экрана (localhost вместо IP)
🧹 System Maintenance
Компиляция приложений, очистка кэша, оптимизация ART
  • cmd package compile -r -a
    Android 7+ Full Recompile: Перекомпиляция всех приложений в speed-mode. Ускоряет запуск, но занимает время.
    -r = reset
    -a = all
  • cmd package compile -m speed-profile -f -a
    Recommended Профильная компиляция (только hot-path). Баланс скорости и памяти.
  • cmd package bg-dexopt-job
    Фоновая оптимизация DEX (выполняется обычно ночью). Полезно запустить вручную после обновления системы.
  • cmd package trim-caches 9999M
    Агрессивная очистка кэша всех приложений (freespace 9999MB — триггер для очистки всего).
  • pm compile --reset -a
    Сброс всех ART-компиляций (если система стала нестабильной после оптимизации).
🎨 UI: Анимации & DPI
  • settings put global window_animation_scale 0.5
    Скорость анимации окон (0.5 = x2 быстрее, 0 = off)
  • settings put global transition_animation_scale 0.5
  • settings put global animator_duration_scale 0.5
    Длительность аниматора (Material Design анимации)
  • wm density 400
    Android 6+ Изменение DPI (плотности экрана). 400 = компактный интерфейс. wm density reset для сброса.
  • wm size 1080x1920
    Принудительное разрешение (полезно для тестирования или фиксации aspect ratio).
  • settings put system font_scale 1.0
    Масштаб шрифта (1.0 = 100%, 0.9 = 90% — компактнее).
🔋 Doze & Battery
Doze mode доступен с Android 6.0+. На Android 12+ использует App Standby Buckets.
  • dumpsys deviceidle enable
    Root/Shell Принудительное включение Doze mode (глубокий сон).
  • dumpsys deviceidle whitelist +com.whatsapp
    Добавить приложение в whitelist Doze (не будет убиваться в фоне). Вместо WhatsApp подставьте нужный пакет.
  • dumpsys deviceidle force-idle
    Debug Принудительно перевести в Deep Doze (для тестирования).
  • dumpsys deviceidle unforce
    Debug Выход из принудительного Deep Doze.
  • settings put global app_standby_enabled 1
    App Standby Buckets (Android 9+) — автоматическое ограничение редкоиспользуемых приложений.
  • settings put global forced_app_standby_enabled 1
    Принудительное применение standby даже для активных приложений (aggressive).
  • cmd netpolicy set restrict-background true
    Android 7+ Глобальный запрет фонового использования данных (экономия трафика и батареи).
💾 RAM Optimization
  • am send-trim-memory --user 0 com.android.systemui RUNNING_CRITICAL
    Принудительная очистка памяти для SystemUI (безопасно, пересоздаст интерфейс).
  • cmd stats config disable
    Android 12+ Отключение сбора системной статистики (экономит CPU cycles).
  • cmd activity memory-factor critical
    Симуляция критической нехватки памяти (система агрессивно убьет фоновые процессы).
  • settings put global ram_expand_size 4294967296
    Android 11+ Установить zRAM = 4GB (4294967296 bytes). Требует перезагрузки. Не работает на Samsung.
📲 App Control
  • cmd appops set com.facebook.katana RUN_IN_BACKGROUND ignore
    Android 7+ Запретить конкретному приложению работать в фоне (неубиваемый аналог для любого Android).
  • cmd appops set com.whatsapp RUN_ANY_IN_BACKGROUND allow
    Разрешить приложению работать в фоне без ограничений (аналог "Закрепить").
  • cmd shortcut reset-all-throttling
    Android 8+ Сброс лимитов на создание ярлыков (если приложение больше не может добавлять ярлыки).
  • cmd jobscheduler dump com.google.android.gms
    Просмотр активных job для GMS (найдите нужный ID перед запуском).
  • am force-stop com.android.vending
    Принудительная остановка Google Play Store (убить процесс).
  • pm disable-user --user 0 com.package.name
    No Root Отключение bloatware для текущего пользователя (без root).
  • pm enable com.package.name
    Включение ранее отключённого приложения.
  • pm install-existing --user 0 com.package.name
    Восстановление системного приложения, если оно было удалено (не для всех OEM).
🔐 Permission Control
  • cmd appops set com.package POST_NOTIFICATION ignore
    Android 7+ Полное отключение уведомлений для приложения (на уровне appops).
  • pm revoke com.package android.permission.CAMERA
    Отзыв конкретного разрешения у приложения.
  • pm grant com.package android.permission.WRITE_SECURE_SETTINGS
    Выдача разрешения WRITE_SECURE_SETTINGS (нужно для SetEdit и некоторых твиков).
  • cmd appops set com.package READ_CONTACTS deny
    Жёсткий запрет на чтение контактов (даже если пользователь разрешил в настройках).
🎮 Input & Hardware
  • input keyevent 26
    Эмуляция нажатия кнопки питания (блокировка/разблокировка экрана).
  • input keyevent 82
    Эмуляция кнопки Menu / Recent Apps.
  • input keyevent 4
    Эмуляция кнопки Back.
  • input swipe 300 1000 300 200
    Свайп снизу вверх (x1 y1 x2 y2). Полезно для автоматизации.
  • input tap 500 500
    Тап по координатам экрана.
  • cmd statusbar expand-notifications
    Программное раскрытие шторки уведомлений.
🏭 OEM Specific
Xiaomi / HyperOS
  • settings put system miui_home_animation_rate 0
    Отключение анимации лаунчера MIUI (MIUI 13 и ниже)
  • settings put secure speed_mode_enable 1
    Режим скорости (Settings.Secure)
  • am kill com.miui.powerkeeper
    Caution Мягкая остановка PowerKeeper (менее агрессивное управление батареей). Не использовать force-stop!
OnePlus / OOS
  • settings put global game_mode_status 1
    Включение Game Mode глобально
  • settings put global oneplus_screen_refresh_rate 2
    OOS 12- Фиксация 120Hz (2 = высокая частота). На OOS 13+ работает через display.refresh_rate.
Samsung / OneUI
  • settings put global game_performance 4
    OneUI 4.x- Game Performance Mode: 4 = максимум. На OneUI 5+ ключ изменился.
  • settings put global adaptive_battery_management 0
    Отключение адаптивного управления батареей
🧼 Logs & Privacy
  • dumpsys dropbox set rate low
    System Отключение сбора логов крашей (экономит storage и CPU). На Xiaomi может сбрасываться.
  • logcat -c
    Root Очистка буфера логов (требует root на Android 10+).
  • dmesg -c > /dev/null
    Root Очистка kernel ring buffer (требует root).
  • settings put global sys_vm_stats 0
    Запрет на отправку usage stats (эффект зависит от OEM).
📱 Version Specific
  • settings put global cached_apps_freezer enabled
    A12+ Заморозка кэшированных приложений (App Freezer).
  • settings put global app_standby_enabled 1
    A9+ App Standby Buckets.
  • settings put global ram_expand_size 4294967296
    A11+ ZRAM size в bytes (4GB). Требует перезагрузки.
  • settings put global forced_app_standby_enabled 1
    A9+ Принудительный standby.
  • cmd shortcut reset-all-throttling
    A8+ Shortcuts API.
